# HelloWorld

Simple repository containing a Hello World program.

## Contents

- A minimal program that prints "Hello, World!". The language may vary; add or replace files as needed.

## Run

Pick the file for your language and run with the appropriate command. Examples:

- Python:

	python3 hello.py

- Node.js:

	node hello.js

- Java:

	javac HelloWorld.java
	java HelloWorld

If you don't have a program yet, create a file named `hello.py` with:

```python
print("Hello, World!")
```

# HelloWorld

Simple repository containing one or more "Hello, World!" example programs.

## Contents

- Minimal example programs that print "Hello, World!" in various languages. Add or replace files as needed.

## Run

Choose the file for your language and run with the appropriate command. Examples:

- Python:

	python3 hello.py

- Node.js:

	node hello.js

- Java:

	javac HelloWorld.java
	java HelloWorld

If you don't have a program yet, create a file named `hello.py` with this content:

```python
print("Hello, World!")
```

Then run:

```bash
python3 hello.py
```

1. Make your changes locally.

